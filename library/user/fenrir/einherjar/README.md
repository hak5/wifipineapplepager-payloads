## Overview

EINHERJAR enables coordination between multiple WiFi Pineapple Pagers for distributed attacks. Using Bluetooth Low Energy (BLE) for discovery and communication, EINHERJAR allows one Pager to act as the **Commander** while others become **Warriors** that execute orders.

## Requirements

- Multiple WiFi Pineapple Pagers (2+)
- BLE adapter on each Pager (hci0)
- hcitool and bluetoothctl installed
- BLE range: ~10-30 meters

## Usage

### On Commander Pager:
1. Run EINHERJAR payload
2. Select "Commander" or "Auto-Detect"
3. Scan for warriors
4. Issue commands from the menu

### On Warrior Pagers:
1. Run EINHERJAR payload
2. Select "Warrior" or "Auto-Detect"
3. Wait for commands from Commander
4. Press button to exit
